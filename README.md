# Queen's Call - Bot Permanente em Chamadas

### Sobre o Projeto
Bot desenvolvido para permanecer conectado em chamadas de voz automaticamente (ex: servidores do Discord). Ideal para manter canais abertos ou acompanhar atividades em call.

### Como Usar
1. Crie um bot no Portal do Desenvolvedor do Discord (https://discord.com/developers/applications)
2. Copie o token do bot e cole no arquivo config.json
3. Instale as dependências com: `pip install discord.py`
4. Execute o bot com: `python bot_call.py`

### Tecnologias Utilizadas
- Python
- discord.py (biblioteca para interagir com a API do Discord)
